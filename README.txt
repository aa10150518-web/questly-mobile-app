紙藝Quest 手機 App 網站版

玩法：
每天完成紙藝作品、發文、跟 AI 聊設計點子、回覆留言、收藏靈感，都會累積今日分數。
分數會推進紙藝地圖、開寶箱、拿掉落物，讓真實創作行動變成闖關。

上傳方式：
1. 把這整個 questly-mobile-app 資料夾上傳到 Netlify、GitHub Pages 或其他靜態網站服務。
2. 手機打開網站網址。
3. iPhone：Safari 分享按鈕 -> 加入主畫面。
4. Android：Chrome 選單 -> 新增至主畫面 / 安裝應用程式。

注意：
- PWA 安裝與離線快取需要 https 網址；直接打開本機檔案不會啟用。
- 入口檔是 index.html。
